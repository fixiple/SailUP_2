# SailUP is using : 
- Angular 17.0.8
- Node 20.10.0
- npm 9.1.3
- Nest JS 10.2.1

## To launch backend : 

1) use these commands below

```console
cd back
npm run start
```

2) navigate to http://localhost:3000/


## To launch frontend :

1) use these commands below

```console
cd front 
ng serve
```

2) navigate to http://localhost:4200/
